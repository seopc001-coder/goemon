## 🛠️ SEOツール推奨セットアップ

### 必須ツール

#### 1. Google Analytics (GA4)
**設定手順:**
```html
<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```

**追跡イベント:**
- ページビュー
- CTAクリック
- スクロール深度
- セクションビュー
- LINE誘導クリック

#### 2. Google Search Console
**設定手順:**
1. [https://search.google.com/search-console](https://search.google.com/search-console) にアクセス
2. プロパティを追加（URLプレフィックス）
3. 所有権の確認（HTMLタグ方式）
4. サイトマップの送信

**確認タグ:**
```html
<meta name="google-site-verification" content="あなたの確認コード" />
```

#### 3. Google Tag Manager (GTM)
**メリット:**
- タグの一元管理
- コード変更不要でトラッキング追加
- イベントトラッキングの柔軟性

### 推奨ツール

1. **Ahrefs** または **SEMrush**
   - キーワードリサーチ
   - 競合分析
   - 被リンク分析
   - 順位追跡

2. **Screaming Frog SEO Spider**
   - サイトクロール
   - テクニカルSEO監査
   - メタデータチェック

3. **GTmetrix** / **PageSpeed Insights**
   - ページ速度測定
   - パフォーマンス改善提案

4. **Ubersuggest**
   - 無料キーワードツール
   - コンテンツアイデア
   - 競合分析
