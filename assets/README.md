# Assets フォルダ - 必要な画像・動画ファイル

このフォルダには、LPで使用する画像や動画ファイルを配置します。

## 📁 必要なファイル一覧

### 必須ファイル

#### 1. ヒーロー動画
**ファイル名**: `hero-video.mp4`
- **推奨サイズ**: 1920x1080px
- **フォーマット**: MP4 (H.264コーデック)
- **フレームレート**: 30fps以下
- **ファイルサイズ**: 10MB以下推奨
- **長さ**: 10-20秒（ループ再生）
- **内容案**:
  - ナイトワークの店舗の雰囲気
  - SNSアイコンやグラフのアニメーション
  - スマホでTwitterを操作している様子
  - 輝くネオンエフェクト

**取得方法:**
- フリー動画サイト: Pexels, Pixabay, Videvo
- AI動画生成: Runway, Pika Labs
- 制作依頼: ビデオグラファーに依頼

---

#### 2. OGP画像（SNSシェア用）
**ファイル名**: `og-image.jpg`
- **サイズ**: 1200x630px（必須）
- **フォーマット**: JPG または PNG
- **ファイルサイズ**: 1MB以下
- **内容**:
  - Re:ラボのロゴ
  - キャッチコピー: 「ナイトワークの"これから"をデザインする」
  - サービス名: 「Twitter運用代行」
  - 視覚的な要素: ネオンエフェクト、グラフなど

**テンプレート:**
```
+------------------------------------------+
|                                          |
|        [Re:ラボ ロゴ]                    |
|                                          |
|     ナイトワークの"これから"を          |
|           デザインする                  |
|                                          |
|      Twitter運用代行サービス            |
|                                          |
|   [ネオンエフェクト / グラフ装飾]        |
|                                          |
+------------------------------------------+
```

---

#### 3. Favicon
**ファイル名**: `favicon.png`
- **サイズ**: 32x32px または 64x64px
- **フォーマット**: PNG（透過背景推奨）
- **内容**: Re:ラボのロゴマーク（シンプル版）

---

#### 4. LINE QRコード
**ファイル名**: `line-qr.png`
- **サイズ**: 500x500px以上
- **フォーマット**: PNG
- **取得方法**:
  1. LINE公式アカウント管理画面にログイン
  2. 「友だち追加」→「QRコード」をクリック
  3. QRコードをダウンロード

**設置場所**:
- `index.html` の `.qr-placeholder` クラス内に配置

## 📝 実装手順

### ステップ1: 画像の準備
1. 上記リストの必須ファイルを用意
2. 推奨サイズに合わせてリサイズ
3. ファイル名を指定通りに設定
4. このフォルダに配置

### ステップ2: HTMLファイルの更新
動画ファイルを追加後、必要に応じてパスを確認:
```html
<video class="hero__video" autoplay muted loop playsinline>
    <source src="./assets/hero-video.mp4" type="video/mp4">
</video>
```

QRコードを追加後:
```html
<div class="qr-placeholder">
    <img src="./assets/line-qr.png" alt="LINE QRコード">
</div>
```

### ステップ3: 最適化
1. 画像を圧縮（TinyPNG等）
2. WebP形式への変換（オプション）
3. Lazy Loadingの確認

## 🆘 トラブルシューティング

### 動画が再生されない
- ファイル形式を確認（MP4推奨）
- コーデックを確認（H.264推奨）
- ファイルサイズを削減
- 別のブラウザで確認

### 画像が表示されない
- ファイルパスを確認
- ファイル名の大文字小文字を確認
- ブラウザキャッシュをクリア

### ページ速度が遅い
- 画像を圧縮
- WebP形式を使用
- CDNを導入検討
